Project Overview

Purpose

The AI Content Detection Tool provides a preliminary method for identifying AI-generated content in written texts such as academic papers, assignments, and professional documents.

Key Objectives

  • Support authenticity and originality in academic work
  • Flag content that appears machine-generated
  • Reduce need for extensive human review
  • Uphold academic and professional standards
  • Ensure submitted work reflects genuine human effort

Target Users

  • Academic Institutions: Universities and lecturers
  • Researchers: Academic and research bodies
  • Project Stakeholders: Development team and supervisors

SMART Goals

Backend Detection Engine

Build an NLP-based engine that identifies AI patterns with confidence scores (0-100%). Trackable through database analysis results.

File Upload Feature

Enable text-only file uploads with 80% success rate using Flask. Track successful database insertions.

File Validation

Block 100% of invalid uploads through MIME-type checks. Verify no invalid files create database records.

Analysis Logging

Log all analyses (document name, score, date) to SQLite database with 100% completeness during testing.

Documentation

Create comprehensive documentation totaling at least 10 complete pages, tracked via shared platform.
